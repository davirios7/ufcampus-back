## Linguagem utilizada

- TypeScript

## Requisitos

- **Pnpm:** 9.4.0 ou superior

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/davirios7/ufcampus-back
```

Entre no diretório do projeto

```bash
  cd ufcampus-back
```

Instale as dependências

```bash
  pnpm install
```

Inicie o projeto

```bash
  pnpm run dev
```

Servidor hospedado na porta 3001.
